Struct RustString

Represents raw parts of a Rust String, encoded in UTF-8. Content in ptr does not need to be null-terminated and can contain embedded null character.

Remarks

Do not forget to wrap and export the Marshal.FreeHGlobal method (currently you have to do it manually. If you want your library to be more accessible with the corresponding rhodopsin Rust crate, export it with the signature void ffi_free(size_t pointer)) and call it after finishing using the string. Forgetting doing so might cause memory leak. DO NOT try to free the pointer using the free function or equivalent in the FFI-bridged code. The pointer is allocated using .NET NativeAOT's allocator that is independent from the allocator of the FFI-bridged code. Doing so will trigger an UB and will almost always cause unpredictable chaos.
